Source Switzerland from natural earth data

How to source Switzerland from natural earth data?

Before we start, please get GDAL installed

  1. Download 图包. On this page，Choose “Admin 1 - States, Provinces“ 。

    Why? Admin 1 shows internal borders of each state or province. You can choose "Admin 0 - Boundary lines"

  2. Unzip it.

  3. Now please use og2ogr to get switzerland geojson out:

    ogr2ogr -f GeoJSON -where "ADM0_A3 in ('CHE')" switzerland.geojson ne_10m_admin_1_states_provinces.shp

  4. Now we can open it inside geojson.io:
